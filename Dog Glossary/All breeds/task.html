Description

Now that we've shown sub-breeds, let's show all breeds with the sub-breeds.

When the user clicks the new button, show all breeds as a list. You can check the API documentation for the endpoint.

Start by adding a new button with the id of button-show-all and the text Show All Breeds.

The functionality part has the same constraints as before, so display the data as an ordered list.

Also, if there is a sub-breed for a breed, show the sub-breeds as an unordered list inside the list item.

You can use the IDs we've provided to select the elements.

Objectives

Your page should:

  • Contain a button with the ID button-show-all and the text Show All Breeds;
  • Use the DOG API;
  • Display all breeds as an ordered list and sub-breeds as a nested unordered list from the API.
